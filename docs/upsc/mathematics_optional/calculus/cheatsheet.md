## Integration

- $\int \frac{x - \sqrt{1 - x^2}}{(1 - x^2)^2}e^{\frac{x}{\sqrt{1 - x^2}}}dx$

  Put $x = sint$ and integration would become easy.

  Similarly do same substitution in $\int \frac{1}{(1 - x^2)^{3/2}}dx$

- $\int cosec(2x)dx = log(tanx)/2$
